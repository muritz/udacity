HTML Basics

Elements of the web

The web consists of users, computers, servers, HTTP, and the internet
  • HTTP(hypertext tranfer protocoll) the main protocol of the web
  • Servers are the computers hosting the files that make up the web
  • Internet is the largest computer network
  • Browser is a program on the computer which displays files found on the web by interpreting their code

HTML tags are used to create HTML documents

Basic HTML elemts consist of an opening tag and closing tag and the content in between both.
Void tags are an exception, they do not have a closing tag. We learned about inline (<br>, <span>,...)
and block tags (<div>, <p>, ...) too and their contrast.

HTML-Attributes:

We use them for links and to point at images,like <img src="" alt="">

Struggling: Atom HTML preview is causing atom to crash on my macOS Sierra 10.12.6

I used the Live Help in Udacity, a very nice guy. He recommened me to reinstall. I replied: I did
twice already! Then i asked if it's enough to just dump the "atom.app" into the trash. He simply
wrote, "you have to do more than that" and provided me with the link below. I could feel him smiling.
He knew obviously that i'd be surprised about what i didn't know.
Finally i deleted all the installation, tried twice again and then decided to skip Atom's HTML preview.

Here is the link to the corresponding atom.io discussion forum.
After dumping the atom.app, i had to delete all these files as shown below.
  • ~/.atom
  • /usr/local/bin/atom
  • /usr/local/bin/apm
  • /Applications/Atom.app
  • ~/Library/Preferences/com.github.atom.plist
  • ~/Library/Application Support/com.github.atom.ShipIt
  • ~/Library/Application Support/Atom
  • ~/Library/Saved Application State/com.github.atom.savedState
  • ~/Library/Caches/com.github.atom
  • ~/Library/Caches/Atom
